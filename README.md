# 3D Intelligent Traffic Simulator using OpenGL

This project is a real-time 3D intelligent traffic simulation system developed using Python OpenGL.

## Features

- Realistic 3D urban traffic environment
- Lane-aware vehicle movement
- Smooth acceleration and deceleration
- Safe distance and collision avoidance
- Manual and automatic traffic signal control
- Day/night mode
- Lamp posts active at night
- Crash effect with smoke and vehicle removal
- First-person/follow camera mode
- Interactive keyboard and mouse controls

## Technical Constraint

This project uses only the OpenGL/GLUT/GLU functions allowed in the provided lab assignment template.  
No external graphics engine, physics engine, shader system, or additional OpenGL utility library is used.
